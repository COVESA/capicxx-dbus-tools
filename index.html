Welcome to CommonAPI C++ for D-Bus!

CommonAPI C++ is a C++ framework for interprocess and network communication which defines a high-level C++ API that can be used for different IPC mechanisms such as D-Bus. Please visit http://genivi.github.io/capicxx-core-tools/ if you are new to CommonAPI C++.

In principle the CommonAPI C++ D-Bus binding is based on the standard D-Bus library (http://www.freedesktop.org/wiki/Software/dbus) but needs some patches of libdbus which are published together with the runtime code of the D-Bus binding.

Basic Features

  • High-performance serialization of D-Bus messages including introspection.
